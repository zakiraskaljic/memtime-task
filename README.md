# Memtime task

## Overview

Manage clients, projects, tasks, and time entries. Navigate clients, their projects, and tasks as a hierarchy. For each task, you can view associated time entries and create, update, or delete them. You can also see detailed information for clients, projects, tasks, and individual time entries.

## Features

- **Navigation:** Top menu with links to Clients and Time Entries
- **Hierarchical View:** Navigate clients → projects → tasks
- **Details View:** See detailed information for clients, projects, tasks, and time entries
- **Task Management:** Manage time entries associated with tasks (create, update, delete)
- **Time Tracking Overview:** See all time entries across the system in one place

## Installations

- npm install
- npm run dev
